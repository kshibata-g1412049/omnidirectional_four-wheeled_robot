# omnidirectional_four_wheel_robot
This repository is a URDF file that describes an omnidirectional four-wheeled robot for ROS users.

## ROS 2 (Jazzy + Gazebo / gz) version

A ROS 2 port lives alongside the original ROS 1 (catkin) files under
[`ros2/omnidirectional_four_wheeled_robot/`](ros2/omnidirectional_four_wheeled_robot/).
The ROS 1 files in the repository root are left unchanged.

See [`ros2/omnidirectional_four_wheeled_robot/README.md`](ros2/omnidirectional_four_wheeled_robot/README.md)
for build and launch instructions.
